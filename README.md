# 🌊 一色海岸 24h マリンダッシュボード

葉山・一色海岸に特化した、24時間先までの気象・海洋データを一元表示する自分専用ダッシュボード。
ブラウザのCORS制限や海快晴のログイン壁を回避するため、Node.js (Express + Playwright) のローカルサーバー構成です。

## データソース
- **Yahoo!天気（葉山町）** … 天気・気温・降水量（cheerioでHTMLパース）
- **海快晴（一色海岸）** … 波・風・潮位・潮回り（Playwrightで自動ログイン）
- **Open-Meteo** … Marine API(波) + ECMWF(風)（Windy相当の数値）

## 機能
- 上部に今日の潮回り（潮名・満潮/干潮の時刻と潮位）とアドバイス
- 24時間先までを毎時で比較する統合タイムラインテーブル
- 固定列（天気・気温・降水量・潮位）＋ タブ切替（風／波／波風の比較）
- 強風5m/s以上のハイライト、一色海岸の地形を考慮したオン/オフショア判定
- 満潮/干潮バッジ、潮位トレンド、深夜帯の視認性調整

## セットアップ
```bash
# 1. 依存パッケージをインストール（Playwrightのchromiumも自動取得）
npm install

# 2. 環境変数を設定
cp .env.example .env
#   → .env を開き、海快晴のログインID/パスワードを入力

# 3. 起動
npm start
```
ブラウザで http://localhost:3000 を開く。
スマホで見る場合は、同じWi-Fi内で `http://<MacのIP>:3000`（例: http://192.168.11.9:3000）。

## 注意
- `.env` には海快晴のパスワードが入るため、Gitには**コミットしない**（`.gitignore`で除外済み）。
- Macがスリープ中・Wi-Fi圏外・オフラインでは動作しません（ローカルサーバーのため）。

## 構成
| ファイル | 役割 |
|---|---|
| `server.js` | スクレイピング＆API（`/api/weather`） |

| `package.json` | 依存・起動スクリプト |
